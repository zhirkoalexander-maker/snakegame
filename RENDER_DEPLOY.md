# Деплой на Render.com (самый простой)

1. Откройте: https://render.com/
2. "Sign Up" через GitHub (один клик)
3. "New" → "Web Service"
4. "Connect repository" → выберите: zhirkoalexander-maker/snakegame
5. Настройки:
   - Name: snake-multiplayer
   - Root Directory: server
   - Build Command: npm install
   - Start Command: npm start
6. "Create Web Service"
7. Через 2 минуты получите URL

URL будет вида: `https://snake-multiplayer.onrender.com`
